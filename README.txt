Devuelve una lista de SKUs, inventario y precios de una manera liviana y eficiente para la sincronización con Omnicommerce.

== Description ==

Omnicommerce es una plataforma de sincronización de tiendas e-commerce con sistemas de gestión ERP y punto de venta actualmente disponible sólo para Argentina.

Este plugin compila una lista de códigos de artículos (SKU), inventario y precios de manera liviana y eficiente,
con ello permitiendo sincronizaciones más rápidas y estables con menor impacto en el servidor.

A tener en cuenta:

* Este plugin funciona sólo al ser llamado por Omnicommerce
* No es funcional por sí mismo para la sincronización
